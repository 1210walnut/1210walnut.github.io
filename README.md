Nao’s Website

個人用ポートフォリオサイトです。

概要

HTML・CSS・JavaScriptのみで作成したシンプルなホームページです。

以下の機能を実装しています。

* レスポンシブ対応
* アニメーション付きタイトル
* 背景パーティクルエフェクト
* スクロール演出
* ホバーアニメーション
* プロフィール紹介
* スキル紹介
* 作品紹介

インストール

リポジトリをクローンします。

git clone https://github.com/yourname/my-homepage.git

フォルダへ移動します。

cd my-homepage

index.html をブラウザで開いてください。
